# Filter menu on the right edge of the streams view is cut off

When the filter dropdown for a column of the rightmost column is opened, if there are enough columns to fill the width of the viewport, the filter dropdown is cut off because it flows off the right side of the page. In these cases, it should shift left so that its right edge lines up with the page instead of being partially hidden by the page edge.
